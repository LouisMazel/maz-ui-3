useWait #

Vue composable to handling wait plugins

WARNING

You must install wait plugin before use it

TIP

More info about wait plugin in its documentation

Usage #

vue
<template>
  <Loader v-if="wait.isLoading('MAIN_LOADER')" />
  <p v-else> Loaded </p>
</template>

<script lang="ts" setup>
  import { useWait, sleep } from 'vue'

  const { wait } = useWait()

  onMounted(() => {
    wait.start('MAIN_LOADER')
    await sleep(2000)
    wait.stop('MAIN_LOADER')
  })
</script>
