Maz-UI

Library of standalone components and tools for Vue & Nuxt (v3.x)

Getting Started Components Plugins Helpers Filters Directives 
npm install maz-ui

yarn add maz-ui

Standalone

All components or tools are standalone, if you want use only one module from this library you don't need to install the whole library

SSR Friendly

All components works with Nuxt, no need to install components on client side

Typescript

This library is writing with Typescript, all types and declarations are directly availables

Made with TailwindCSS

Optimized CSS file sizes | All CSS good practices are respected with TailwindCSS | Use your theme easily

Dark and Light Theme

All components supports the Dark and Light themes.

Tools

Maz-ui provide many modules: plugins, directives, filters and helpers
