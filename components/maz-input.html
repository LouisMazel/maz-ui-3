MazInput â

MazInput is a standalone component that replaces the standard html input text with a beautiful design system. Many options like colors, sizes, disabled, loading, error, warning, valid states, error messages and icons are included.

INFO

Before you have to import the global css files in your project, follow instructions in Getting Started

Basic usage â

vue
<template>
  <MazInput
    v-model="inputValue"
    label="Label"
  />
</template>

<script setup>
  import { ref } from 'vue'
  import MazInput from 'maz-ui/components/MazInput'

  const inputValue = ref('value')
</script>

Options â

Password â


html
<MazInput
  v-model="inputValue"
  label="Label"
  type="password"
/>

Placeholder â


html
<MazInput
  placeholder="placeholder"
/>

Required â

Will add * charac to label and placeholder

html
<MazInput
  label="label required"
  required
/>

Disabled â


html
<MazInput
  v-model="inputValue"
  label="label disabled"
  disabled
/>

Hint â

Will replace the label, usefull to show error message

html
<MazInput
  v-model="inputValue"
  label="label hint"
  hint="An error occured"
  error
/>

Icons â

INFO

When you use right-icon or left-icon, the component use MazIcon

Use your own icons

html
<MazInput
  v-model="inputValue"
  label="label icons"
  left-icon="banknotes"
  right-icon="user"
/>

<!-- or -->

<MazInput v-model="inputValue" label="label icons">
  <template #left-icon>
    <MazIcon name="check" class="maz-text-xl" />
  </template>
  <template #right-icon>
    <MazIcon name="user-group" class="maz-text-xl" />
  </template>
</MazInput>

Auto focus â

Will focus automatically the component

html
<MazInput
  v-model="inputValue"
  label="label auto-focus"
  auto-focus
/>

No radius â

Will remove the border radius

html
<MazInput
  v-model="inputValue"
  label="label no-radius"
  no-radius
/>

Debounce â

The value emit by the input will be delayed, usefull for searching

input value: value

html
<!-- 3000ms = 3 secondes -->
<MazInput
  v-model="inputValue"
  label="label debounce"
  debounce
  :debounce-delay="3000"
/>

Sizes â

Use the attribute size with a value in mini, xs, sm, md, lg, xl

vue
<template>
  <MazInput
    v-for="size in sizes"
    :key="size"
    v-model="inputValue"
    :label="size"
    :size="size"
  />
</template>

<script setup>
  const sizes = ['mini', 'xs', 'sm', 'md', 'lg', 'xl']
</script>

Colors â

TIP

Click on each input to show colors

INFO

Use the attribute color with a value in this list, the component will use this color

vue
<template>
  <MazInput
    v-for="{ name } in colorsArray"
    :key="name"
    v-model="inputValue"
    :label="name"
    :color="name"
  />
</template>

<script setup>
  const colors = {
    primary: { name: 'primary', hex: '#1e90ff' },
    secondary: { name: 'secondary', hex: '#1cd1a1' },
    info: { name: 'info', hex: '#17a2b8' },
    success: { name: 'success', hex: '#9acd32' },
    warning: { name: 'warning', hex: '#fcb731' },
    danger: { name: 'danger', hex: '#ff6d6a' },
    white: { name: 'white', hex: '#fff' },
    black: { name: 'black', hex: '#000' },
  }
</script>
